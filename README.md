# MMA Weight Calculator 🥊

A simple and efficient Python tool designed for fighters to calculate their weight classes, manage weight cuts, and keep track of their performance safely. Built as part of my coding and physical training journey.

---

## 🚀 Features
* **Weight Class Detection:** Instantly find out which official MMA weight class you belong to.
* **Fighter-Focused:** Built specifically with the mindset of discipline and combat sports.
* **Terminal Interface:** Clean, fast, and lightweight CLI execution.
